## About the Project 
The Online Ticket Booking Platform (Client) is the frontend application of a ticket booking system built with React. It focuses on clean UI, smooth navigation, and responsive design to provide users with an easy and enjoyable ticket booking experience.

Users can search for tickets, view detailed information, authenticate securely, and complete bookings through an intuitive interface.

---

## Project Overview  
Objective:
To build a user-friendly and modern UI for an online ticket booking system.

Target Users:
Users who want to search tickets, check details, and book tickets online quickly and easily.

---

## Key Features  

=> Browse available tickets
=> View ticket details (date, destination, price, description)
=> User authentication using Firebase
=> Ticket booking interface
=> Fully responsive design (mobile, tablet, desktop)
=> Smooth animations using Framer Motion
=> Toast notifications and alerts for better UX

## Installation️ & Setup
1. Clone the repo and install dependencies:

```bash
git clone https://github.com/Nusrat-Islam/online_ticket_booking_client.git
cd online_ticket_booking_client
npm install
```

2. Run the application:

```bash
npm run dev
```
